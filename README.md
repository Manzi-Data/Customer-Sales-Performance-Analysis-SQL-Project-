&#x20;**Customer \& Sales Performance Analysis (SQL Project)**



&#x20;  **Overview**

This project simulates an e-commerce company (NovaRetail) and analyzes business performance using SQL.



1\. **Objectives**

\- Identify top customers

\- Analyze product profitability

\- Evaluate employee performance

\- Measure marketing ROI

\- Track sales trends



2\. **Tools Used**

\- SQL Server

\- T-SQL



3\. **Key Skills**

\- Joins

\- Aggregations

\- Window Functions (RANK, LAG)

\- CTEs

\- Data Modeling



4\. **Key Insights**

\- Revenue is concentrated among top customers

\- High-margin products drive profitability

\- Sales show seasonal patterns

\- Some campaigns are not profitable



5\. **Project Structure**

\- schema.sql → database structure

\- analysis\_queries.sql → SQL analysis

\- insights.md → business insights



&#x20;**Conclusion**

This project demonstrates how SQL can be used to solve real-world business problems and generate actionable insights.
